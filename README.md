## Technologies

## Run Locally

Clone the project

```bash
  git clone git@github.com:Tonips22/Portfolio.git
```

Go to the project directory

```bash
  cd Porfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
